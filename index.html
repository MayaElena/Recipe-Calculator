Welcome to the UArizona Garden Kitchen Nutritional Calculator!

Whether this is your first time here, or you've clicked the help button, let us explain how the Nutritional Calculator works!

Basics

This nutritional calculator takes in your ingredients, and calculates the calories and essential nutritional information for your needs against the FDA Daily Values.

How To Use

First, please wait for the application to load. This will look like a red spinner over the center of the page, and the ingredients input will be greyed out.


Once the spinner has vanished, please proceed!


Next, you will notice the table has three options, 'Ingredient', 'Quantity', and 'Unit.' At first, only 'Ingredient' will be inputtable. This is on purpose, so that we can find you the write units of measurement and nutritional information


Now, input an ingredient into the 'ingredient' column and hit 'Enter' on your keyboard, or click off of the input. If we have seen this ingredient before, we will automatically load it. If we haven't, that's okay too! If it's unknown to use, we will display a modal popup with a few options:

  • Fuzzy Logic Match: If the input wasn't easily recognized or fragmentary, we will try to match it with our natural language processing.
  • Misspelling Match: If the input was misspelled, we will try to match it with our learned ingredients
  • Load New Ingredient: If the input wasn't recognized, we will use our API's ability to learn and search new ingredients to add it to our database!

Now you can input the units of measurement, i.e. cups, ozs, etc. and the quantity as well


Add more ingredients with the 'Add Ingredient to Recipe' button in tools, and once you are happy with your recipe, click 'Calculate Recipe'

Warnings

This calculator comes with a series of warnings if it detects something wrong:

Daily Value

Should any of the daily values (Calories, Fat, etc) be exceeded per serving, the calculator will warn you, and display a red tooltip on the side of the nutritional facts

Ingredients

If an ingredient is found that is exceedingly high in those daily values, or represents a tremendous amount of those daily values as a percentage, it will be flagged in the ingredients table as well

Garden Kitchen Recommendations

Additionally, the UArizona Garden Kitchen recommends a healthy diet, and if any of the Garden Kitchen's Recommendations are exceeded, a pop up will display with which recommendations have been exceeded.
